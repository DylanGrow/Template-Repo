# ⚡ Modern Web Template

A high-performance, professional boilerplate for static web projects. Built for speed, organized for scale, and ready for automated deployment.

## 🚀 Quick Start
1. Click **"Use this template"** to create a new repo.
2. Edit `index.html` and `src/css/style.css`.
3. Commit changes to see them live via GitHub Pages.

## 🛠️ Features
- **Auto-Deploy:** Pre-configured GitHub Actions for instant hosting.
- **Modern CSS:** CSS variable system with built-in Dark Mode support.
- **Magic Favicon:** SVG icon that adapts to browser themes.
- **Pro Structure:** Clean separation of concerns (assets, src, tests).

## 📂 Project Structure
- `src/`: Main source code (JS/CSS/Data).
- `assets/`: Media, fonts, and icons.
- `.github/`: CI/CD workflows and PR templates.
